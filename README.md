# Webpack starter

This is webpack starter enviroment for the web development.

Setup includes:

- JavaScript ES Modules
- SCSS with convenient folder structure and ready to use function, breakpoints and imports for confort front end development
- Images and fonts loaders

To use this setup

    Clone master branch: git clone https://github.com/VaheAA/webpack-setup.git
    Run npm install
    Run dev server: npm start
    Dev build: npm run build:dev
    Production build: npm run build:prod
